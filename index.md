# Ciência de Dados Aplicada ao Direito

**Autores:** Ivar Hartmann e Bruno da Cunha de Oliveira

## Sobre este livro

Este livro apresenta os fundamentos de ciência de dados aplicada ao contexto
jurídico brasileiro. O material cobre desde os fundamentos básicos de programação
em Python até técnicas de análise de dados, processamento de linguagem natural
e métodos estatísticos aplicados ao direito.

## Estrutura

### Parte I: Fundamentos de Programação
1. **Os Fundamentos da Programação** — Variáveis, condicionais, iteração, listas e dicionários
2. **Automação de Tarefas Jurídicas Repetitivas** — Funções, map e zip

### Parte II: Dados e Análise
3. **Grande Volume de Dados** — Estatística descritiva, bibliotecas e Pandas
4. **Análise de Linguagem Natural** — Expressões regulares e edição de documentos

### Parte III: Fontes de Dados e Incerteza
5. **Fontes Adicionais de Dados Jurídicos** — Web scraping
6. **Lidando com a Incerteza** — Amostras, probabilidade e curva normal
